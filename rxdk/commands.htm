Commands

RXDK commands are available in the Command Palette (all prefixed RXDK:) and, for most, as rows in the RXDK sidebar. Some are gated until prerequisites are installed or a debug session is active.

Setup & project

CommandDoes
RXDK: Complete SetupInstall all prerequisites (.NET, SDK, docs, Zig, host tools).
RXDK: New ProjectScaffold a project from a template (wizard).
RXDK: New Prebuilt XBE ProjectCreate a prebuilt-XBE project from an existing .xbe.
RXDK: Refresh Source FolderRe-sync a prebuilt project's source view.

Build / deploy / run

CommandDoes
RXDK: BuildCompile + link + imagebld → .xbe, then packs an .iso (staging any deployPaths alongside it).
RXDK: DeployCopy the .xbe and deployPaths to xe:\<name>\.
RXDK: RunLaunch the deployed title.
RXDK: Debug (F5)Build, deploy, launch under the debugger, attach symbols.

Device

CommandDoes
RXDK: Set Xbox IP / HostnameSet the target console address.
RXDK: Launch xbWatsonOpen the standalone debug-string monitor.

Debugging

CommandDoes
RXDK: Cycle Globals VisibilityAdvance the Globals pane filter: Title → Title + constants → All. See Debugging. (Available during a debug session.)

Docs, SDK & settings

CommandDoes
RXDK: Extension DocumentationOpen this documentation set.
RXDK: Xbox SDK DocumentationOpen the Xbox SDK API reference.
RXDK: Open SDK Folder (include/lib)Reveal the staged SDK folder.
RXDK: Fetch Latest RXDK-SDKClone/pull the latest SDK.
RXDK: Install .NET 8 RuntimeDownload & install the runtime.
RXDK: SettingsOpen the RXDK Settings screen.

See also: Getting started · Settings.
