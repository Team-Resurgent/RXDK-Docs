Settings

RXDK settings live under the rxdk.* and xbox.* namespaces in VS Code settings. The most common ones are also grouped in the RXDK Settings screen.

RXDK Settings screen

A curated panel that groups the settings you reach for most, with a batched Apply / Cancel (unlike native settings, which apply instantly). Open it from:

  • The Settings (gear) row in the RXDK sidebar, or the gear in the sidebar title bar.
  • RXDK: Settings in the Command Palette.

Sections:

SectionFields
DebuggingGlobals visibility, Reveal title output
DeviceDefault Xbox IP / hostname
ProjectsDefault parent folder for new projects

Changes save to your User (global) settings on Apply. Changing Globals visibility here refreshes a running debug session live.
